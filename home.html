Projects

News Room - Spring 2016
An android news app which offers top stories news along with weather information for the current location. Logged in users can write and save notes on Firebase similar to google keep. They can also chat with other users. Material Design features and advanced animations like Flip View and List Buddies used. Also libraries like Retrofit and Google location services used.

Movie Flix - Spring 2016
Android app designed and developed to learn and implement some of the features of Android, like Recycler View, Async Task. Built a RESTful server to store the details of the different movies. Using PHP got the data from the server in JSON format for access in the app (connection between the model and the view).

Seasons - May 2016
Designed and developed an android app to provide weather conditions and forecasts using real-time weather data. Retrofit library used for interacting with APIs and sending network requests with OkHttp.

Notes - June 2016
Developed a note taking app similar to Google Keep. Advanced features like Recycler view, Material Design, Floating Action button used. Allows a user to create, read, update and delete a note. Storing of the notes is done in a SQLite database.

Web based Job Notifications System - Fall 2015
Built a web application to post new jobs and then send those notifications to subscribed users using Java (Servlets and JSP). The web app allowed a user to subscribe to receive notifications when a new job is posted matching his skills and also allowed the admin to post new jobs for the user to view.

Remote Code Management Facility - Spring 2015
Designed and implemented a facility for Remote Code Management using C++. This software facilitated transferring of files to and from the local and remote machines, exploring the directory subtree on the remote machine. Socket Based message-passing communication channel was designed and implemented for communication.

Movie recommendations for users using Map Reduce - Spring 2015
Developed a movie recommendation application for users based on Netflix dataset using MapReduce. Recommends top 5 movies to the user in the genre he/she likes based on his/her rating history.

The Story Maker - Spring 2015
Built a website and web service. Supported creating stories where a story is a sequence of story blocks laid out on a time line using HTML5, CSS, JavaScript and ASP.NET MVC.

Laser Height Measurement Indication System using 8051 - June 2014
Signal conditioner and display system built and programmed to convert and display the output obtained in terms of current from the laser sensor, in order to get the height for the pilots to see during testing purposes.

Flight Data Monitoring System using ARM Cortex M3 - February 2014
Developed a program to display the flight data on flight, in real time, to the flight test engineer, obtained from ACRA KAM-500, to check for any errors during testing.
